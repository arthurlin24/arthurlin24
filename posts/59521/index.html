Introduction

It’s been quite some time since I’ve aspired to create a personal blog and begin sharing stories from my own life. Having no technical background should never be an obstacle, especially in this digital age.

I used to wonder what I should write and how to approach it, but now I’ve come to realize that there’s value in sharing the ordinary moments of my life.

Among the three popular static site generators – Jekyll, Hugo, and Hexo – I opted for Hexo because it offers the simplest and most straightforward approach, allowing me to concentrate on creating content.

What is Hexo?

Hexo is a fast, simple and powerful blog framework. You write posts in Markdown (or other markup languages) and Hexo generates static files with a beautiful theme in seconds.

How to install Hexo

Essentially, all you need to do is follow the guidance provided in the Hexo documentation. Before setting up Hexo, you’ll need to install Git and Node.js, both of which are straightforward processes. Once you’ve completed these prerequisites, you’ll be all set to get started.

Then you can run Hexo using hexo <command>.
